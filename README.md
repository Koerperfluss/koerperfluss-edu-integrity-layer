# Educational Integrity & Accessibility Framework (EIAF)
**Part of the Körperfluss AI Middleware for Medical Education**

This framework provides a standardized logic layer to ensure **hallucination-free AI interactions** and **digital accessibility (WCAG 2.2 / BFSG)** in academic environments.

### Core Features:
- **Educational Guardrails:** Logic for Interleaved RAG to validate AI outputs against verified clinical guidelines and academic standards.
- **Accessibility Layer:** Automated transformation of complex technical language into "Plain Language" (Leichte Sprache) and metadata for screen readers.
- **LTI 1.3 Integration Patterns:** Open-source patterns for connecting AI-driven reasoning agents to Moodle 5.0.

### Why this matters:
AI in medical education must be 100% reliable. This framework allows institutions to implement AI tutoring without risking "Capability Dilution" or accessibility barriers. It is designed to support 10,000+ students in the DACH region.
